Golang API Testing


Getting Started
This project build using Golang, Gin Gonic. This project give an example how to create API Unit Test in Golang.

Prerequisites
Golang go1.10.4
Gin Gonic https://github.com/gin-gonic/gin
Testing Plugins https://github.com/stretchr/testify

Running the tests
go test -v

Built With
Golang
Gin Gonic
